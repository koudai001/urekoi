# テーブル定義

## User

- `ID` BIGSERIAL PK
- `EMAIL` VARCHAR(255) NOT NULL UNIQUE
- `PASSWORD` VARCHAR(255) NOT NULL（ハッシュ）
- `CREATED_AT` TIMESTAMPTZ NOT NULL DEFAULT now()
- `UPDATED_AT` TIMESTAMPTZ NOT NULL DEFAULT now()

## Profile

- `ID` BIGSERIAL PK
- `USER_ID` BIGINT NOT NULL UNIQUE FK → User.ID (1:1)
- `NICKNAME` VARCHAR(50) NOT NULL
- `AGE` SMALLINT NOT NULL
- `PREFECTURE_CODE` SMALLINT NOT NULL FK → Prefecture.ID
- `INTRODUCTION` TEXT
- `CREATED_AT` TIMESTAMPTZ NOT NULL DEFAULT now()
- `UPDATED_AT` TIMESTAMPTZ NOT NULL DEFAULT now()

## ProfileImage

- `ID` BIGSERIAL PK
- `PROFILE_ID` BIGINT NOT NULL FK → Profile.ID
- `URL` VARCHAR(255) NOT NULL
- `SORT_ORDER` SMALLINT NOT NULL　表示順、長押しで入れ替え可能にする想定
- `CREATED_AT` TIMESTAMPTZ NOT NULL DEFAULT now()
- `UPDATED_AT` TIMESTAMPTZ NOT NULL DEFAULT now()

## Prefecture

- `ID` SMALLINT PK（JIS X 0401コード 1〜47）
- `NAME` VARCHAR(10) NOT NULL（都道府県名 例: 東京都）

## Like

- `ID` BIGSERIAL PK
- `FROM_USER_ID` BIGINT NOT NULL FK → User.ID
- `TO_USER_ID` BIGINT NOT NULL FK → User.ID
- `CREATED_AT` TIMESTAMPTZ NOT NULL DEFAULT now()

## Match

- `ID` BIGSERIAL PK
- `USER1_ID` BIGINT NOT NULL FK → User.ID
- `USER2_ID` BIGINT NOT NULL FK → User.ID
- `MATCHED_AT` TIMESTAMPTZ NOT NULL DEFAULT now()

## Message

- `ID` BIGSERIAL PK
- `MATCH_ID` BIGINT NOT NULL FK → Match.ID
- `SENDER_USER_ID` BIGINT NOT NULL FK → User.ID
- `BODY` TEXT NOT NULL
- `CREATED_AT` TIMESTAMPTZ NOT NULL DEFAULT now()
