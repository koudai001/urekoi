# API設計

## 認証

| メソッド | エンドポイント  | 説明     |
|----------|-----------------|----------|
| POST     | /signup         | 新規登録 |
| POST     | /login          | ログイン |
| POST     | /logout         | ログアウト |
| DELETE   | /users/me       | 退会     |

## プロフィール

| メソッド | エンドポイント | 説明               |
|----------|----------------|--------------------|
| GET      | /profile       | 自分のプロフィール取得 |
| POST     | /profile       | プロフィール作成   |
| PUT      | /profile       | プロフィール更新   |

## 相手の検索

| メソッド | エンドポイント      | 説明         |
|----------|---------------------|--------------|
| GET      | /profiles           | 相手一覧取得 |
| GET      | /profiles/{id}      | 相手詳細取得 |

## いいね

| メソッド | エンドポイント    | 説明               |
|----------|-------------------|--------------------|
| POST     | /likes            | いいねを押す       |
| GET      | /likes/received   | もらったいいねを取得 |

## マッチング

| メソッド | エンドポイント | 説明                   |
|----------|----------------|------------------------|
| GET      | /matches       | マッチングした相手の一覧取得 |

## メッセージ

| メソッド | エンドポイント                        | 説明                         |
|----------|---------------------------------------|------------------------------|
| GET      | /matches/{match_id}/messages          | 特定ユーザーとのメッセージ履歴を取得 |
| POST     | /matches/{match_id}/messages          | 特定ユーザーにメッセージを送る |
