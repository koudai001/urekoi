# API設計

## 認証

| メソッド | エンドポイント  | 説明     |
|----------|-----------------|----------|
| POST     | /signup         | 新規登録 |
| POST     | /login          | ログイン |
| POST     | /logout         | ログアウト |
| DELETE   | /users/me       | 退会     |

### POST /signup

リクエスト

```json
{
  "email": "test@example.com",
  "password": "password123"
}
```

レスポンス (201 Created)

```json
{
  "id": 1,
  "email": "test@example.com"
}
```

| ステータス | 条件                       |
|------------|----------------------------|
| 201        | 登録成功                   |
| 400        | バリデーションエラー(email形式・password 8文字未満など) |
| 409        | email重複                  |

### POST /login

リクエスト

```json
{
  "email": "test@example.com",
  "password": "password123"
}
```

レスポンス (200 OK)

```json
{
  "token": "<JWT>"
}
```

| ステータス | 条件                                   |
|------------|------------------------------------------|
| 200        | ログイン成功                              |
| 400        | バリデーションエラー                      |
| 401        | email未登録 or パスワード不一致(列挙攻撃防止のため区別せず統一) |
